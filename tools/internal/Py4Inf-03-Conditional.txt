::Ερώτηση 3 - multiple choice, no shuffle::[html]
Ποια από τις παρακάτω προτάσεις είναι αληθής για το παρακάτω τμήμα προγράμματος:
<pre>
if  x == 5 :
    print('Is 5')
    print('Is Still 5')
    print('Third 5')
</pre>
{
= Ανάλογα με την τιμή του x, είτε θα εκτελεστούν και οι τρεις συναρτήσεις print είτε καμία.
~ Το αλφαριθμητικό 'Is 5' θα εκτυπώνεται πάντοτε, ανεξαρτήτως της τιμής του x.
~ Το αλφαριθμητικό 'Is 5' δεν θα εκτυπωθεί σε καμία περίπτωση, ανεξαρτήτως της τιμής του x.
~ Αν η τιμή του x είναι μικρότερη του μηδενός θα εκτελεστούν μόνο δύο από τις τρεις συναρτήσεις print.
}

::Ερώτηση 5 - multiple choice, shuffle::[html]
Βλέπετε το παρακάτω τμήμα κώδικα:
<pre>
if x == 6 :
    print('Is 6')
    print('Is Still 6')
    print('Third 6')
</pre>
Φαίνεται τέλειο, αλλά η Python σας εμφανίζει ένα 'Indentation Error' (Δφάλμα Εσοχής) στη δεύτερη print. Ποιος είναι ο πιο πιθανός λόγος;
{
= Έχετε χρησιμοποιήσει tab και κενά στο αρχείο.
# Φροντίστε να βρείτε την επιλογή αυτόματης επέκτασης των tab, στο πρόγραμμα επεξεργασίας κειμένου σας. Διαφορετικά, θα είναι πολύ ενοχλητικό όταν θα εμφανίζονται αυτά τα σφάλματα σε κώδικα που *φαίνεται* τέλειος.
~ Η Python πιστεύει ότι το 'Still' είναι μια λάθος λέξη στη συμβολοσειρά.
~ Η Python έχει φτάσει στο όριό της του μεγαλύτερου προγράμματος, που μπορεί να εκτελεστεί.
~ Για να κάνει τους ανθρώπους να αισθάνονται ανεπαρκείς, η Python εμφανίζει τυχαία 'Indentation Errors' σε απολύτως σωστό κώδικα - μετά από περίπου μία ώρα το σφάλμα θα εξαφανιστεί χωρίς καμία αλλαγή στο πρόγραμμά σας.
}

::Ερώτηση 7 - multiple choice, shuffle::[html]
Τι θα εκτυπώσει ο παρακάτω κώδικας;
<pre>
x = 0
if x < 2 :
    print('Small')
elif x < 10 :
    print('Medium')
else :
    print('LARGE')
print('All done')
</pre>
{
= Small
All done
~ Small
Medium
LARGE
All done
~ Medium
All done
~ Small
}

::Ερώτηση 8 - multiple choice, shuffle::[html]
Στον παρακάτω κώδικα:
<pre>
if x < 2 :
    print('Below 2')
elif x >= 2 :
     print('Two or more')
else :
    print('Something else')
</pre>
Ποια τιμή του 'x' θα προκαλέσει την εκτύπωση του 'Something else';
{
= Ο κώδικας αυτός δεν θα εκτυπώσει σε καμία περίπτωση 'Something else', ασχέτως της τιμής του 'x'.
# Δεν θα εκτυπωθεί ποτέ επειδή όλες οι τιμές για το 'x' είναι είτε κάτω από 2 είτε μεγαλύτερες ή ίσες με δύο. Άρα θα εκτελεστεί είτε το if είτε το elif αλλά ποτέ η περίπτωση else.
~ x \= 2
~ x \= 2.0
~ x \= -2
}

::Ερώτηση 9 - multiple choice, shuffle::[html]

(1)   astr = 'Hello Bob'

(4)   astr = '123'
(5)   istr = int(astr)

</pre>
{
= 1
~ 3
~ 6
~ 2
}
