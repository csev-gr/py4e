::Ερώτηση 9 - multiple choice, shuffle::[html]
Ας υποθέσουμε ότι η μεταβλητή x έχει αρχικοποιηθεί σε μια ακέραια τιμή (π.χ. x = 3). Τι κάνει η παρακάτω εντολή;
<pre>
x = x + 2
</pre>
{
= Ανακτά την τρέχουσα τιμή του x, προσθέτει δύο σε αυτό και βάζει το άθροισμα ξανά στο x
~ Τερματίζει το πρόγραμμα
~ Δεν θα εκτελεστεί καθώς πρόκειται για συντακτικό σφάλμα
~ Δημιουργεί μια συνάρτηση που ονομάζεται "x" και βάζει την τιμή 2 στη συνάρτηση
}
